# Jamazon
This application implements a simple command line based storefront using Node.js and MySQL. 
There are three interfaces available in this application:  customer, manager, and supervisor.

## Technologies Used
1.	Javascript
2.	nodeJS
3.	MySQL
4. 	npm packages:
    -	mysql
    -	dot-env
    -	inquirer
    -	colors/safe
    -	cli-table
